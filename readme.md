Provide a name in the URL fragment or type a name to query Wikidata and report likely pronouns.

[Demo](https://jncraton.github.io/pronouns/)

## Usage

Open `index.html` in a browser or visit the demo link. Append `#Name` to the URL to automatically look up pronouns for "Name".

## Testing

Run tests using `make test`. This requires `uv` and `playwright`.
